# Introduction

元銀行員バックエンドエンジニア。

大学生の頃に彼女の誕生日に HTML/CSS で90年代レベルのサイトを作って祝う。これがプログラミングの原体験。人生で初めて、能動的に勉強をした。

しかし、それが好きであるとか、仕事にしたいという感覚はまるでなく、文系であったので、就職先から除外。

銀行に新卒で入社。ひとつも楽しくなく、真に楽しいと思える仕事をしようと思い、プログラミングをふと思い出す。

一念発起し、1年2ヶ月で退社、JavaScript を用いた開発を担当。

自動 UI テストは Java(Selenide)。

半年後、Laravel を用いた Web アプリケーションの開発も合わせて担当。

現在は Go で to B 向け API サーバーの構築を行っている
